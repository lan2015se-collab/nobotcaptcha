# 歡迎來到NobotCAPTCHA的GitHub專案

## 介紹

我們是NobotCAPTCHA。

我們的網站（nobot-captcha.lovable.app）是在 Lovable 這個以AI 為核心的程式開發代理，協助你用自然語言打造全端應用程式，

我們發現，

有許多的詐騙集團試圖使用 Lovable 、 Netlify 和 Vercel 製作詐騙釣魚網站來當作釣魚網站的入口

我們的團隊（Aethe.gp）於2026/02/27決定製作一款真正免費的防機器人的驗證碼應用程式

## 特點

1.完全免費使用所有功能

2.非詐騙釣魚網站

3.簡單的驗證框/文字驗證

4.精準的人機驗證

5.後台分析+快速加入到你的網站/應用

## 如何使用

1.進入網站
[NobotCAPTCHA官方網站](nobot-captcha.lovable.app) 

2.按下 免費開始使用 按鈕並完成註冊

3.登入後按下左上角按鈕找到 API密鑰 並按下

4.輸入你要使用NobotCAPTCHA的網站網址按下 新增 按鈕

5.依照網站給的專屬程式碼指令貼到您的網站內
